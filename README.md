# Design6_labs

these are the labs that are required for the 6th itertion of the design spine at Stevens.
The course goes over many essential engineering centric ideas and thought patters that include: 
- Design Analysis
- Needs Assessment 
- Structruing the Search for the Problem
- Acquiring Applying and Protecting Technical Knowledge
- Synthesis 
- Ethics
- Hazards Analysis
- Implementation 

Along with these ideas, Design6 also covers labs that introduce studnets to important technical topics that will be useful in the implementation of their final project. These labs can be found on Professor Lu's github: https://github.com/kevinwlu
